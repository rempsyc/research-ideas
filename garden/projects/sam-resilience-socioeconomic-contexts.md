# SAM resilience across socioeconomic contexts

This project examines self-administered mindfulness interventions across socioeconomic contexts. The public OSF title is "Building Resilience Across Socioeconomic Contexts: A Multi-Site Study on Self-Administered Mindfulness (SAM) Interventions."

Main group: [[sparacio-lab]].

## Public materials

- OSF project: [Building Resilience Across Socioeconomic Contexts](https://osf.io/7mnqd/overview)

## Related context

The OSF project is forked from a larger public project on self-administered mindfulness interventions. The broader research context includes large multisite work testing brief standalone mindfulness exercises for stress reduction.
