# Nonfiction meta-reanalysis

This project revisits the evidence base around nonfiction reading and psychological outcomes. It is a bridge between the book intervention work and the broader question of what kinds of written materials produce measurable change.

The immediate work is data cleaning, recoding, and producing a first re-analysis that can become the backbone of a manuscript.

Main group: [[van-bavel-lab]].

## Current status

- Data cleaning and recoding are in progress.
- First re-analysis and first manuscript draft are the next milestones.
